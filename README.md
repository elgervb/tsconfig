# elgervb Stylelint config

## installation

```
npm install elgervb-stylelint-config
```
or
```
yarn install elgervb-stylelint-config
```

## usage

Add the following to your `tslint.json` file

```
{
  "rulesDirectory": [
    "node_modules/elgervb-tslint-config"
  ],
}
```

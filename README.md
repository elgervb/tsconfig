# elgervb TSLint config

## installation

```
npm install tslint-config
```
or
```
yarn install tslint-config
```

## usage

Add the following to your `tslint.json` file

```
{
  "rulesDirectory": [
    "node_modules/tslint-config-elgervb"
  ],
}
```
